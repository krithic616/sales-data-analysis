 Sales Data Analysis Project

 Overview
 
This project analyzes sales data using Python to identify trends and generate simple business insights.

 Objective
 
To analyze historical sales data and understand sales trends and regional performance in order to support better business decision-making.

 Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

 What This Project Does
- Cleans and prepares the dataset  
- Converts order dates into monthly format  
- Analyzes monthly sales trends  
- Compares sales performance by region  
- Identifies top 10 products by total sales  

 Files in This Repository
- Sales_Data_Analysis.ipynb — Complete analysis notebook  
- train.csv — Source dataset  

 Key Insights
- Sales show steady growth with seasonal patterns  
- West and East regions generate the highest sales  
- Top 10 products contribute a large portion of total revenue
